

🎞️ Python アニメーションエディタ Pro
Python Animation Editor Pro は、複数の静止画から直感的な操作でアニメーションを作成し、GIF形式で出力できるデスクトップアプリケーションです。

🌟 概要
「描いた絵をすぐに動かしたい」というニーズに応えるための、軽量なパラパラ漫画制作ツールです。画像のインポート、順序の入れ替え、再生速度の微調整、そして最終的なGIF書き出しまで、エンジニアリングの効率を重視したUIで設計されています。

✨ 特徴機能
直感的なタイムライン編集:

リストボックスによるコマ（フレーム）の管理。

ボタン一つでコマの順序を「上へ」「下へ」自由に変更可能。

リアルタイム・プレビュー:

指定したミリ秒（ms）単位でアニメーションをループ再生。

スライダーで再生速度を動的に変更し、動きの質感を即座に確認。

スマート・インポート:

複数画像を同時選択して一括読み込み。

アスペクト比を維持しつつ、エディタに最適なサイズ（400x400）へ自動リサイズ。

GIFエクスポート機能:

設定した順序・速度を保持したまま、高品質なGIFアニメーションとして保存。

🛠 技術スタック
GUI: tkinter

Image Processing: Pillow (PIL)

Logic: Pythonicなデータ管理（List of Dictionaries）による高速なフレーム制御

🚀 クイックスタート
1. 依存ライブラリのインストール
Bash
pip install Pillow
2. アプリの起動
Bash
python flipbook_editor.py
🕹 使い方
画像を追加: 「画像を追加」ボタンから連番画像などを選択します。

順序を調整: 右側のリストでファイル名を選択し、「↑」「↓」ボタンでアニメーションの順序を整えます。

速度を調整: 下部のスライダーを動かして、最適な再生速度（ms）を探ります。

GIF保存: 「GIF保存」ボタンで、完成したアニメーションを書き出します。

📂 クラス設計
FlipbookEditor:

image_data: 画像のバイナリとプレビュー用データを分離して管理し、メモリ効率を確保。

animate: root.after を利用した再帰的呼び出しによる、非ブロッキングな再生ロジック。
